スクリーニング実行

1

インデックスを選択

ページ上部のタブ（日経225 / グロース250 / S&P 500 / NASDAQ 100）で対象市場を選ぶ。

2

「スクリーニング実行」をクリック

全銘柄のデータをYahoo Financeから取得し、複合スコアを計算する。日経225で約3〜5分かかる。

3

結果は自動保存される

再読み込みやページ遷移後も前回の結果を自動復元。「上位 N 銘柄」セレクタで表示数を変更可能（10/20/30/50）。

モメンタムスコア（100点満点）

6指標のパーセンタイルランクを加重平均した総合スコア。スコアが高いほど「勢いがある」銘柄。

指標 内容 ウェイト
1ヶ月リターン直近1ヶ月の価格上昇率20%
3ヶ月リターン直近3ヶ月の価格上昇率20%
出来高比5日平均出来高 ÷ 20日平均出来高15%
MA乖離率株価と50日移動平均線の乖離15%
MACDヒストグラムトレンドの強さ・方向性15%
RSI14日相対力指数15%
ウェイトプリセット: 「バランス型 / 出来高重視 / トレンド重視 / 初動特化 / リバーサル」の5種から切替可能。スクリーニング済みデータを即座に再計算する。

分析タブ一覧

スクリーニング完了後に表示されるタブ。データがある場合のみ表示される。

モメンタム 日本株・米国株 共通

モメンタムスコア上位銘柄の一覧テーブル。列ヘッダークリックでソート可能。

主な表示列

  • スコア モメンタムスコア（100点満点）
  • 状態 過熱 / GC / 本命 / 52W / BB圧 / NEW など
  • SQ スクイーズスコア（米国株のみ）
  • 1M / 3M 1ヶ月・3ヶ月リターン
  • RS1M / RS3M ベンチマーク対比の相対強度

状態バッジの意味

  • 過熱 RSI > 70 の過熱圏
  • GC ゴールデンクロス発生
  • 本命 RS 判定で最強クラス
  • 52W 52週高値ブレイクアウト
  • NEW 前回から新規ランクイン
  • ↑+N 前回比スコア急上昇
  • OBV↑乖離 出来高が価格上昇を先行示唆
  • OBV↓乖離 出来高が価格下落を先行示唆
  • ADX25 トレンド強度25以上（明確なトレンド）
  • W9EMA↑ 週足9EMA準拠中（米国株のみ）
  • W9EMA割れ 週足9EMA下抜け（米国株のみ）
逆張り候補 日本株・米国株 共通

アナリスト目標株価との乖離・PER割安度・RSI売られすぎを複合して「割安な逆張り候補」を抽出。

  • • 逆張りスコア: 目標株価乖離率(40%) + 推奨(15%) + PER割安(15%) + EPS成長(15%) + RSI(15%)
  • • アナリスト推奨は strong_buy → 強い買い 形式で日本語表示
セクター回転 日本株・米国株 共通

1Mリターン（縦軸）× 3Mリターン（横軸）のバブルチャートで資金の流れを4象限で可視化。

右上: 加速

中期も短期も強い。資金が流入中

左上: 回復

中期弱・短期強。底打ちの兆候

右下: 減速

中期強・短期弱。天井圏の可能性

左下: 衰退

中期も短期も弱い。資金が流出中

セクター相関 日本株・米国株 共通

直近60営業日の日次リターンに基づくセクター間の相関係数をヒートマップで表示。

  • • 正の相関（赤）: 同じ方向に動きやすい → 分散効果が低い
  • • 無相関（グレー）: 独立した動き → 分散効果が高い
  • • 負の相関（青）: 逆方向に動きやすい → ヘッジ効果あり
  • • セクター内に3銘柄以上ある場合のみ表示
ブレイクアウト 日本株・米国株 共通

52週高値に近い銘柄・ボリンジャーバンド圧縮銘柄を自動検出して専用テーブルに表示。

  • • 52W新高値: 現在値が52週高値の99%以上
  • • BB圧縮: ボリンジャーバンド幅が閾値以下（爆発前の収束状態）
  • • 52W乖離: 52週高値からの距離（マイナスほど高値圏）
品質マトリクス 日本株・米国株 共通

モメンタムスコア（横軸）× 品質スコア（縦軸）の4象限チャート。右上が「高モメンタム×高品質」の理想銘柄。

  • • 品質スコア: ATR%・上下出来高比・ギャップ率・ヒゲ比率から算出
  • • エントリー難易度: 良好 / 押し待ち候補 / 追いかけ注意 / ボラ高注意 / 様子見 の6段階
  • • タブ内の「データソース健全性」でデータ取得状況を確認できる
種まき度 日本株専用

機関投資家が短期業績悪化で売った「種まきフェーズ」銘柄を財務指標から検出。

  • • タイム裁定候補: 設備投資急増(≥+20%) × 一過性赤字 × 営業CFプラス
  • • 小型優良株: 時価総額100〜3000億円 × 高モメンタムスコア
  • • 種まき度スコアは設備投資・CF黒字・売上成長・利益減・株価失望の5要素から算出
米国株分析 米国株専用

4つのシグナルを加重合成した US Advanced Score（55点以上の銘柄を一覧表示）。

シグナル 内容 ウェイト
EPS改定予想EPS vs 実績EPS → 上方/下方/横ばい30%
機関投資家フロー機関保有比率の変化 → 買い増し/売り減らし30%
決算後ドリフト決算後の価格継続性 → 上昇継続/下落継続25%
オプション動向yfinance option_chain()から GEX・PCR・IV Rank・MaxPain を算出15%

市場レジーム

騰落比率・ADL・モメンタムスコア分布・セクターローテーションを統合し、現在の相場状態を自動分類する。

🟢

強気 (Bull)

広範な上昇。モメンタム戦略が最も有効。

🔴

弱気 (Bear)

広範な下落。守備的な対応を推奨。

🟡

中立 (Neutral)

方向感なし。個別銘柄の選別が重要。

🔵

分散 (Mixed)

セクター間で乖離。ローテーション戦略が有効。

「信頼度」はシグナルの一致度を示す。35%以下は判断材料として参考程度に。

日次レポート

スクリーニング毎に自動生成。レジーム × ブレス × セクターローテーション × 変化検知を総合して候補を3分類する。

🌱 初動候補

新規ランクイン・52W高値圏・RS急上昇の銘柄

💪 継続強者

高スコアかつRS本命を継続している銘柄

⚠️ 過熱注意

RSI > 70 または過熱シグナルが点灯している銘柄

📋 モーニングブリーフ

スクリーニング完了時に自動生成される市場サマリーカード。チャットドロワー（🤖ボタン）を開くと先頭に表示される。

  • • 地合いバッジ: 現在の市場レジーム（強気/弱気/中立/分散）
  • • サマリー行: 騰落比率・資金流入セクター・新規ランクインなどのハイライト
  • • 注目銘柄バッジ: 初動候補（紫）/ 継続強者（水色）を色分け表示
  • • 過熱注意バッジ: RSI高水準・過熱シグナル点灯銘柄をローズ色で警告
  • • WLアラート: ウォッチリスト銘柄のスコア急変・カスタムアラート発火を表示
  • • コスト: Gemini 不使用のテンプレート整形 → API コスト 0 円

日本株ページでは日経225+グロース250、米国株ページではS&P500+NASDAQ100のブリーフが表示される。「クリア」ボタンで再表示。

変化通知（🔔）

右上の🔔ボタンにバッジが表示されたら、前回スクリーニングからの変化がある。クリックで引き出しを開く。

  • • 新規ランクイン: 前回圏外だった銘柄が上位に入った
  • • 圏外落下: 前回上位だった銘柄がランク外に落ちた
  • • スコア急上昇: 前回比 +8点以上のスコア上昇
  • • スコア急落: 前回比 −8点以上のスコア低下
  • • カスタムアラート: スマートウォッチリストの条件一致で⚡通知
  • • ウォッチリスト連動: ウォッチリスト登録銘柄の変化を優先通知

銘柄詳細モーダル

テーブルの行をクリックすると詳細モーダルを表示。

  • • モメンタムスコア・品質スコア・エントリー難易度
  • • 種まき度スコア（日本株・スコア20以上の場合表示）
  • • 資本配分スコア（ROE・ROIC・配当性向・自社株買い）
  • • AI自動解説（タグ・投資チェック質問）
  • • サポート/レジスタンス: S1-S3 / R1-R3 の自動検出レベル
  • • テクニカル指標（RSI・MACD・BB・52W高安値）
  • • OBV（スロープ + 乖離判定）、ADX（トレンド強度）
  • • ドローダウン（3M最大DD・現在DD）
  • • ファンダメンタルズ（PER・PBR・EPS・配当利回り）
  • • ショートスクイーズ分析（米国株のみ）
  • • オプション動向（GEX・PCR・IV・MaxPain — 米国株のみ）
  • • モメンタムタイムライン（過去スコア+RSI推移チャート）
  • • スコア内訳グラフ（各指標の貢献度）

📈 モメンタムタイムライン

銘柄詳細モーダルの下部に表示される折れ線チャート。過去のスクリーニングセッションにおけるスコアとRSIの推移を確認できる。

  • • 左軸（紫）: モメンタムスコアの推移
  • • 右軸（オレンジ破線）: RSIの推移
  • • 最大30セッション分の履歴を時系列で表示
  • • スクリーニングが2回以上ある銘柄のみ表示（1回の場合は「データなし」表示）

オプション動向（米国株専用）

銘柄詳細モーダル内に表示。yfinance の option_chain() から直近満期のオプションデータを取得し、7つの指標を算出する。

指標 意味
GEXスコアディーラーのガンマエクスポージャー (-100〜+100)。ネガティブGEX = 価格変動増幅 = ガンマスクイーズリスク
PCR (出来高/建玉)プット/コール比率。1.0未満 = コール優勢（強気）、1.3超 = プット優勢（弱気）
IV Rank現在のインプライドボラティリティの相対水準 (0-100)。80超で「IV高水準」タグ
ATM IVアット・ザ・マネー付近のインプライドボラティリティ (%)
IVスキューOTMプットIV - OTMコールIV。プラス = ヘッジ需要高（フロアプロテクション）
Max Painオプションホルダーの損失が最大になるストライク。SQ前の株価収束先の目安
異常コール出来高コール出来高/プット出来高 > 3 またはコール出来高/OI > 0.5 で検出

🔥 ガンマスクイーズリスク

GEXスコアが -30以下かつ PCR < 0.8（コール優勢）の場合に警戒フラグが点灯。ディーラーがネガティブガンマの状態でコール出来高が集中すると、ディーラーのデルタヘッジ（株買い）が株価上昇をさらに加速させるメカニズム。

ライブAPI（銘柄詳細モーダル）では直近2満期分のデータを使用。US Advanced テーブルの「オプション」列にGEXスコアが表示される。

CF分析（日本株専用）

テーブルの「CF」ボタン、またはモーダルから起動。EDINET DB APIと J-Quants APIを利用。

年次CF（EDINET）

  • • 最大14年の年次CF推移
  • • 営業CF / 投資CF / 財務CF / FCF
  • • M&A余力試算（ネットキャッシュ + FCF × 年数）

四半期CF（J-Quants）

  • • 四半期単体の営業CF推移
  • • J-Quants APIキーが必要（無料プラン対応）

バックテスト

「履歴」→ セッションを選択 → 「バックテスト実行」で過去データのシミュレーションが可能。

  • • 対象期間・上位N銘柄・等加重ポートフォリオを指定して実行
  • • 総リターン・年率リターン・最大ドローダウン・シャープ比を算出
  • • ベンチマーク（SPY / ^N225）との比較エクイティカーブを表示
  • • 結果はDBに保存され、再実行なしで確認可能

⚡ スマートウォッチリスト

ウォッチリスト銘柄にカスタム条件を設定し、スクリーニング実行時に自動チェック → 条件一致で⚡カスタムアラート通知を生成する。

設定方法

  1. ウォッチリストに銘柄を追加
  2. 銘柄詳細モーダルのタイトル横にある ⚡ ボタンをクリック
  3. 指標（RSI, スコア, 1Mリターン等）× 演算子 × 値を設定
  4. 「追加」→「保存」でルールが登録される

利用可能な指標

ウォッチリスト

  • • 各行の「☆」をクリックで登録 / 「★」で解除（削除時は確認ダイアログあり）
  • • 「★ ウォッチリスト」ボタンでウォッチリスト銘柄のみにフィルタ
  • • ウォッチリスト登録銘柄の変化は🔔で優先通知される
  • • インデックスをまたいで登録可能（日経225と米国株を同時管理）

履歴

  • • 「履歴」ボタンで過去のスクリーニングセッション一覧を表示
  • • セッションをクリックするとその時点のランキングを復元
  • • 選択したセッションからバックテストを実行可能
  • • 結果はサーバー再起動後も保持される（SQLite永続化）

銘柄比較モード

テーブルの各行にあるチェックボックスで2〜3銘柄を選択すると、右下に「比較」フローティングボタンが表示される。

  • • 18種類の指標（スコア・RSI・1M/3M・ADX・OBV・DD等）を横並びで比較
  • • 各行で最良の値が緑色にハイライトされる
  • • 「リセット」で選択をクリア

アラートビルダー（⚡）

ツールバーの⚡ボタンから起動。カスタム条件を設定し、スクリーニング結果に自動マッチングする。

  • • 指標（RSI / スコア / 1M / 出来高比 / ADX / 最大DD）× 比較演算子 × 値 を設定
  • • 複数の条件を追加可能（OR条件）
  • • 条件はlocalStorageに保存され、スクリーニング完了時に自動チェック
  • • 一致する銘柄がサマリーカード下に表示される

便利ツール

CSVエクスポート

テーブル右上の「CSV」ボタンでランキングデータをダウンロード。16列（Rank / Ticker / Score / RSI / OBV / DD 等）をExcel対応形式で出力。

列カスタマイズ

テーブル右上の「列」ボタンで表示列を切替。1D / 1W / 1M / 3M / RSI / 出来高比 / RS1M / RS3M の表示/非表示をトグル。設定はブラウザに保存される。

スパークライン

テーブルの銘柄名の横にミニチャートが表示される（sm以上の画面幅）。緑=上昇トレンド / 赤=下落トレンドで一目でパターン把握可能。

スクリーニングETA

スクリーニング実行中にプログレスバーの横に「残り約○分○秒」を表示。直近5回のポーリングレートから移動平均で推定。

ブラウザ通知

スクリーニング完了時にブラウザのNotification APIで通知。別タブで作業中でも見逃さない。初回実行時に通知許可が求められる。

エラーリカバリー

スクリーニングが失敗した場合、エラーメッセージと「再試行」ボタンが表示される。前回のstale状態も自動回復。

カスタマイズ

テーマカラー

右上のカラーピッカーで任意の色を選択。HSLベースでWCAG AA準拠の配色が自動生成される。設定はブラウザに保存される。

ダークモード

右上の ☀/☾ ボタンで切替。設定はブラウザに保存される。

ウェイトプリセット

スクリーニング結果画面で「バランス型 / 出来高重視 / トレンド重視 / 初動特化 / リバーサル」から選択。再スクリーニング不要で即座に再ランク付け。

表示銘柄数

上位 10 / 20 / 30 / 50 銘柄から選択可能。

ソート設定の永続化

テーブルヘッダーをクリックしてソートした設定（キーと方向）はブラウザに保存され、ページ再読み込み後も維持される。

相対強度 (RS) 判定

ラベル 条件 意味
本命 RS1M > +2% かつ RS3M > 0 短期も中期もベンチマーク超え
短期 RS1M > +2% かつ RS3M ≤ 0 短期は強いが中期は平凡
劣後 RS1M ≤ +2% ベンチマーク比で上位にない
テーマ BTC連動など セクター比較が不適切な銘柄

米国株: セクターETF（XLK, XLV, XLF 等11本）比較 / 日本株: 日経平均（^N225）比較
